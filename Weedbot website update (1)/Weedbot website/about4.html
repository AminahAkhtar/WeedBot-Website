    • Biography

    Imtiaz ul Hasan is Research Associate at the Smart City Lab. He is an expert in Artificial Intelligence technologies with more than three years of experience. He has developed and deployed state of the art algorithms for solving complex problems in Healthcare, Agriculture and Environment. His research focuses on machine learning, deep learning, computer vision, object detection and semantic technologies..

    Projects

    His research contributions are reflected in the following projects:

    • Weed detection algorithm for weeding robot: Training and testing of object detection model, using dataset of 18000 images of weed and crops
    • Steering prediction using CARLA: Built a custom Model for steering prediction using Keras
    • Load forecasting using integrated model:Training and evaluation of different machine learning models on the Smart Grid data including Random Forest, Support Vector Machine , Naive Bayes and KNN.
    • Object detection for Autonomous vehicle Training and testing YOLO v5 model for detection of pedestrian, cars and bikes
    • Theme prediction using NLP:Theme classification of text from social media sites in four into different classes.
    • Pet bottle detection for reverse vending machine: Trained MobileNet using transfer learning to classify bottles into 3 different classes including small, big, and nonbottles objects.
